# Glossary — SOCA V1

- **Player**: User showcasing football talent via profile + highlight videos.
- **Scout**: User discovering talent; interacts via discovery + messaging.
- **Club**: Club account; similar discovery needs; may represent an organization.
- **Organization**: Federations/leagues/media/governing bodies; visibility and discovery use-cases.
- **Highlight Video**: A video post uploaded by a player (and used in challenges).
- **Feed**: Public content stream of videos and featured items.
- **Discovery**: Search/filter of players by metadata (position/age/location/role).
- **Explore**: Curated page showing featured players, trending videos, challenges, events.
- **Challenge**: Monthly admin-created contest with submissions and leaderboard.
- **Submission**: A player's entry video for a challenge.
- **Leaderboard**: Ranked list of submissions for a challenge.
- **Exposure / Ranking Signals**: Engagement + scout interactions used to rank content/player visibility.
- **Featured**: Admin/manual highlight for profiles/videos.
- **Report**: A user-submitted flag on content for moderation review.
- **Verification Badge**: Manual badge granted by admin (optional in V1).
